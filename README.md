## Laufzeitumgebung

In der Laufzeitumbebung befinden sich:
    - Eine Datenbank unter localhost:5432, mit Benutzer: "client" und Password "geheim"
    - Eine Serveranwendung unter localhost:8080

## Die Laufzeitumgebung starten

    docker-compose up
    ...
    docker compose down
